A New Era In Scale, Trust, Decentralisation & Fairness In Blockchain.

Graphchain provides a directed acyclic graph (DAG)-based, fully decentralized, consensus mechanism, creating a scalable, fair and simple alternative to current blockchain systems.

How is Graphchain different?

• Build on published research by known cryptographers • Community driven • Open source • Open build • Validator incentives with minimal PoW • Fair-launch • No airdrops • No premine • No nonsense

Features

Fast

Graphchain delivers extremely fast transaction speeds through its novel DAG-based consensus mechanism.

Scalable

Graphchain has a fundamental DAG-like consensus mechanism, which enables exceptional scalability through state updates and agreement times, accommodating huge parallel transaction volumes.

Post-Quantum

Graphchain is built with PQ cryptographic techniques in mind, ensuring best-practice PQ implementations for long-term security based on the current state-of-the-art in research.

Robust

Graphchain uses robust cryptographic techniques for validation, transaction creation, and consensus, incorporating best practices.

Green

Graphchain prioritises energy efficiency and sustainability in consensus, moving away from PoW-heavy approaches.

Checkpointed

Graphchain is designed with implicit and explicit (cross-chain) checkpointing, providing a reliable backup of the current state for recovery and rollback.

Easy

Graphchain has low transaction token costs and quick and simple ways to participate.

Simple

Graphchain has pseudocode, Python, and Rust implementations, making it as human-readable as possible and easy to understand and contribute to.

Fair

No pre-mine or disproportionate initial allocation, ensuring fairness from the start and enabling decentralisation.
