A New Era In Scalability

Graphchain

Graphchain is a collaborative, decentralised, fast and efficient alternative to the classic blockchain. Built to scale.

Graphchain operates a DAG-like consensus mechanism, which vastly improves state update and agreement times.

Features

Fast

Graphchain provides extremely-fast transaction speeds, through the DAG-like consensus mechanism.

Scalable

Graphchain has a fundamental DAG-like design, which enables exceptional scalability, accommodating huge parallel transaction volume.

Robust

Graphchain uses robust cryptographic techniques for validation, transaction creation and consensus.

Green

Graphchain prioritises energy efficiency and sustainability in consensus.

Checkpointed

Graphchain is designed with implicit and explicit (cross-chain) checkpointing, providing a reliable backup of the current state.

Easy

Graphchain has low transaction token costs and quick and simple ways to get participate.
